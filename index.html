Project Ideas

Overview

This project is about developing a 2D game using Unity. The name of this game is “Gem Collection”, it’s a puzzle game with cute user interface and adorable background music. The main goal of each level is to collect enough number of gems in limited steps.

Motivation

The main motivation for me to do this project is that games include many forms of traditional artistic expression, such as illustrations, storyline narrative arcs, and dynamic music. The combination of these artistic expressions can create something that transcends any one type. It is clear that games are where art meets science.

Another motivation is that I have always been interested in game development. This was a great opportunity for me to learn and practice my game development skills through this project.

Description

Tools and Technologie

In order to develop a 2D game, the main and most important equipment is a great computer with hardware that can run Unity on it. The ideally environment would be a Windows computer with operating system Windows 10 with a GPU as we need good quality graphics for the game. We also need an Android phone and an iPhone to test the adaptably of the game to ensure that it can run smoothly on different platforms.

As for software, we are going to need Unity as the main tool to develop the game. Visual studio or Visual Studio Code also needed for scripts developing.
